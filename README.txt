ブラウザからサーバ操作(ディレクトリ移動、コマンド実行、ファイル編集、ファイルアップロード)を行う

PSA
	ディレクトリの指定があるときは現在のディレクトリを変更する
	現在のディレクトリを表示する
	ファイルの指定があるときはファイル名を表示する

アップローダ
	アップロードボタンのクリックでファイルをアップロードする

シェル
	実行ボタンのクリックでコマンドを実行する

ファイルマネージャ
	現在のディレクトリに存在するディレクトリ一覧をリンクとして表示する
		リンクのクリックでそのディレクトリへ移動する

	現在のディレクトリに存在するファイル一覧をリンクとして表示する
		リンクのクリックでそのそのファイルの編集を開始する

エディタ
	ファイルに変更があるときは示す
	保存ボタンのクリックで編集中のファイルを保存する
	行ジャンプのクリックで任意の行にジャンプする
	Tabキーの押下でTabの挿入を行う
	文字列選択時はTabキーの押下でインデントを行う
	文字列選択時はShift + Tabキーの押下でインデントの解除を行う
